# Windows

用于提升在 Windows 系统的个人使用体验。

## 功能

### 安装脚本

- 安装WSL。
- 安装WSL中的各种语言环境，依赖于 Code 项目。
- 安装 AutoHotkey。
- 提供了多类任务的脚本，并加载系统路径中。
- 注册了开启自动启动 AHK 脚本的 Windows 计划任务。
- 提供了一些全流程执行某些任务的 AHK 脚本，同时支持为它们添加 Windows 计划任务。

### PS脚本

略

### 任务脚本

略

### 主 AHK 脚本

- 将指定任务绑定到了自定义的快捷键。

## 使用方法

以管理员身份运行 `Install.ps1` 脚本即可。
